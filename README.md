# Van Tai Danh Nam - Landing Page

Landing page for Van Tai Danh Nam, a local transportation business based in Kien Giang, Vietnam.
Built with pure HTML, Tailwind CSS CDN, and vanilla JavaScript. No build step required.

Live site: https://ihatesea69.github.io/van-tai-danh-nam/

## Services

- Truck freight transport (xe tai cho hang)
- 7-seat contract vehicle rental (xe 7 cho hop dong)
- Private driver service - driver operates customer's own vehicle (tai xe rieng)

## Coverage Area

Rach Gia, Ha Tien, Phu Quoc, Kien Luong, Can Tho, An Giang

## Tech Stack

- HTML5 (single file, no build step)
- Tailwind CSS via CDN
- Swiper.js for service card slider
- AOS (Animate On Scroll)
- Deployed on GitHub Pages

## Deploy

Push to `main` branch. GitHub Pages serves `index.html` from root automatically.
